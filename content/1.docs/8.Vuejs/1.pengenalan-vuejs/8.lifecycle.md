# Modul 8: Lifecycle Hooks

Setiap instance komponen Vue melewati serangkaian langkah inisialisasi saat dibuat. Sepanjang jalan, ia menjalankan fungsi yang disebut **Lifecycle Hooks**, yang memberi pengembang kesempatan untuk menambahkan kode mereka sendiri pada tahap tertentu.

## Diagram Siklus Hidup

Secara garis besar, siklus hidup komponen adalah:
1. **Creation**: Komponen dibuat.
2. **Mounting**: Komponen dimasukkan ke dalam DOM (tampilan browser).
3. **Updating**: Data berubah, dan tampilan diperbarui.
4. **Unmounting**: Komponen dihapus dari DOM.

---

## Hooks yang Sering Digunakan

Dalam Composition API, kita mengimpor hooks dengan awalan `on`.

### 1. `onMounted()`
Dipanggil setelah komponen selesai dirender dan dimasukkan ke dalam DOM. Ini adalah tempat terbaik untuk memanggil API atau inisialisasi library pihak ketiga (seperti grafik/chart).

```vue
<script setup>
import { onMounted } from 'vue'

onMounted(() => {
  console.log('Komponen siap digunakan!')
  // Panggil data dari server di sini
})
</script>
```

### 2. `onUpdated()`
Dipanggil setelah komponen memperbarui pohon DOM-nya karena perubahan status reaktif.

### 3. `onUnmounted()`
Dipanggil setelah komponen dilepaskan/dihapus dari tampilan. Digunakan untuk membersihkan efek samping seperti `setInterval` atau menghapus event listener manual agar tidak terjadi kebocoran memori (*memory leak*).

```vue
<script setup>
import { onMounted, onUnmounted } from 'vue'

let timer

onMounted(() => {
  timer = setInterval(() => {
    console.log('Tik tok...')
  }, 1000)
})

onUnmounted(() => {
  // Penting: Hentikan timer saat pindah halaman!
  clearInterval(timer)
})
</script>
```

---

## Ringkasan Tahapan

| Hook | Waktu Eksekusi | Kegunaan Umum |
| :--- | :--- | :--- |
| **setup()** | Paling awal (sebelum komponen dibuat) | Inisialisasi data & logika. |
| **onMounted** | Setelah elemen HTML muncul di browser | Fetch data, DOM manipulasi. |
| **onUpdated** | Setelah data berubah & DOM diupdate | Reaksi terhadap perubahan UI. |
| **onUnmounted** | Sesaat sebelum komponen hancur | Cleanup, stop timer. |

::callout{icon="i-lucide-alert-circle"}
Jangan pernah memanipulasi DOM secara manual di luar `onMounted`, karena elemen tersebut mungkin belum tersedia di browser.
::
