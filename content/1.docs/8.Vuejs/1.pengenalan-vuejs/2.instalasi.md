# Modul 2: Instalasi & Setup Project

Untuk memulai pengembangan dengan Vue.js secara profesional, kita memerlukan alat bantu pembuatan (*build tools*). Standar modern saat ini adalah menggunakan **Vite**.

## Prasyarat

Sebelum memulai, pastikan komputer Anda sudah terinstal:
1. **Node.js** (Versi 18.x atau lebih tinggi direkomendasikan).
2. **Terminal/Command Prompt**.
3. **VS Code** dengan ekstensi **Vue - Official (Volar)**.

---

## Membuat Project Pertama dengan Vite

Jalankan perintah berikut di terminal Anda:

```bash
# Menggunakan npm
npm create vue@latest

# Menggunakan pnpm (Sangat direkomendasikan untuk kecepatan)
pnpm create vue@latest
```

Anda akan diberikan beberapa pilihan konfigurasi. Untuk tahap belajar, Anda bisa memilih:
- **Project Name**: `belajar-vue`
- **Add TypeScript?**: `No` (Kita mulai dengan JS dulu)
- **Add JSX Support?**: `No`
- **Add Vue Router?**: `Yes`
- **Add Pinia?**: `Yes`
- **Add ESLint?**: `Yes`

---

### Menjalankan Server Development

Gunakan perintah berikut untuk menjalankan aplikasi secara lokal:

```bash
pnpm dev
```

Aplikasi Anda biasanya akan tersedia di `http://localhost:5173`.

---

## Penjelasan `main.js`

File `src/main.js` adalah titik masuk (*entry point*) aplikasi. Di sinilah instance Vue dibuat dan dihubungkan ke DOM:

```js [src/main.js]
import { createApp } from 'vue'
import App from './App.vue'
import router from './router'

const app = createApp(App)

app.use(router)
app.mount('#app') // Menghubungkan ke <div id="app"> di index.html
```

::tip{icon="i-lucide-terminal"}
Jangan lupa untuk selalu memeriksa log terminal jika terjadi error saat menjalankan server development.
::
