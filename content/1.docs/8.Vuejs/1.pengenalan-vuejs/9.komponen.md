# Modul 9: Komponen Lanjutan

Setelah memahami dasar komponen, kita perlu tahu bagaimana komponen saling berkomunikasi.

## 1. Props (Induk ke Anak)
Props adalah atribut khusus yang dapat Anda daftarkan pada sebuah komponen. Data mengalir satu arah dari atas ke bawah.

```vue [Induk: App.vue]
<script setup>
import UserCard from './UserCard.vue'
</script>

<template>
  <UserCard name="Asep" age="25" />
</template>
```

```vue [Anak: UserCard.vue]
<script setup>
// Mendefinisikan props
defineProps({
  name: String,
  age: [String, Number]
})
</script>

<template>
  <div>
    <h3>Nama: {{ name }}</h3>
    <p>Umur: {{ age }}</p>
  </div>
</template>
```

---

## 2. Emits (Anak ke Induk)
Jika anak perlu memberi tahu induk bahwa sesuatu telah terjadi, gunakan event kustom.

```vue [Anak: MyButton.vue]
<script setup>
const emit = defineEmits(['tombolDiklik'])

function notifyParent() {
  emit('tombolDiklik', 'Data dari anak')
}
</script>

<template>
  <button @click="notifyParent">Klik Saya</button>
</template>
```

```vue [Induk: App.vue]
<template>
  <MyButton @tombol-diklik="(val) => console.log(val)" />
</template>
```

---

## 3. Slots (Konten Fleksibel)
Slot memungkinkan Anda mengirimkan elemen HTML sebagai konten ke dalam komponen.

```vue [BaseCard.vue]
<template>
  <div class="card">
    <div class="header">
      <slot name="header">Judul Default</slot>
    </div>
    <div class="body">
      <slot></slot> <!-- Default slot -->
    </div>
  </div>
</template>
```

---

## 4. Provide & Inject (Dependency Injection)
Digunakan untuk mengirim data ke komponen anak yang sangat jauh (deeply nested) tanpa harus melewati setiap level menggunakan props (*prop drilling*).

::card-group
  :::card
  ---
  icon: i-lucide-share-2
  title: Provide
  ---
  Induk menyediakan data.
  ```js
  provide('theme', 'dark')
  ```
  :::

  :::card
  ---
  icon: i-lucide-download
  title: Inject
  ---
  Cicit/Anak mengambil data.
  ```js
  const theme = inject('theme')
  ```
  :::
::
