# Modul 4: Dasar-Dasar Reaktivitas

Salah satu fitur paling ajaib dari Vue adalah sistem reaktivitasnya. Ketika data berubah, tampilan akan diperbarui secara otomatis. Dalam Composition API, kita menggunakan `ref()` dan `reactive()`.

## 1. Menggunakan `ref()`

`ref()` digunakan untuk data dengan tipe primitif (String, Number, Boolean) atau objek yang ingin kita ganti secara utuh.

```vue
<script setup>
import { ref } from 'vue'

const count = ref(0) // Inisialisasi

function increment() {
  // Di bagian script, kita harus menggunakan .value
  count.value++
}
</script>

<template>
  <!-- Di template, kita TIDAK perlu .value -->
  <button @click="increment">{{ count }}</button>
</template>
```

---

## 2. Menggunakan `reactive()`

`reactive()` hanya digunakan untuk objek atau array. Ia membuat objek tersebut menjadi reaktif secara mendalam.

```vue
<script setup>
import { reactive } from 'vue'

const state = reactive({
  user: {
    name: 'John Doe',
    age: 25
  },
  scores: [10, 20, 30]
})

function updateName() {
  state.user.name = 'Jane Doe'
}
</script>
```

::tip{icon="i-lucide-check-circle"}
Gunakan `ref()` secara default untuk hampir semua hal. Gunakan `reactive()` hanya jika Anda memiliki objek besar yang saling terkait dan tidak ingin menulis `.value` berulang kali.
::

---

## 3. Computed Properties

Terkadang kita membutuhkan data yang dihitung berdasarkan data reaktif lainnya. Gunakan `computed()`.

```vue
<script setup>
import { ref, computed } from 'vue'

const firstName = ref('Asep')
const lastName = ref('Surasep')

// fullName akan otomatis terupdate jika firstName atau lastName berubah
const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})
</script>

<template>
  <p>Nama Lengkap: {{ fullName }}</p>
</template>
```

### Mengapa menggunakan Computed?
- **Caching**: Computed hanya akan dihitung ulang jika dependensinya berubah. Jika tidak berubah, ia akan mengambil hasil dari memori (cache), sehingga jauh lebih efisien daripada memanggil fungsi di dalam template.

---

## 4. Watchers

Jika Anda ingin melakukan sesuatu (seperti memanggil API atau menyimpan ke LocalStorage) saat data berubah, gunakan `watch()`.

```vue
<script setup>
import { ref, watch } from 'vue'

const question = ref('')

watch(question, (newVal, oldVal) => {
  console.log(`Pertanyaan berubah dari ${oldVal} menjadi ${newVal}`)
})
</script>
```
