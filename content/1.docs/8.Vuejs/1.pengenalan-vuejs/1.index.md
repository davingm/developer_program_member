# Modul 1: Pendahuluan Vue.js

Selamat datang di Modul 1. Dalam materi ini, kita akan mengeksplorasi apa itu Vue.js, sejarahnya, serta filosofi di balik desainnya yang progresif.

## Apa itu Vue.js?

**Vue.js** (diucapkan /vjuː/, seperti *view*) adalah sebuah framework JavaScript yang bersifat *open-source* yang digunakan untuk membangun antarmuka pengguna (UI) dan *Single-Page Applications* (SPA). Vue diciptakan oleh **Evan You** pada tahun 2014 dengan visi menggabungkan bagian terbaik dari Angular dan React ke dalam sesuatu yang lebih ringan dan mudah dipelajari.

### Filosofi "Progresif"

Vue disebut sebagai **The Progressive Framework**. Artinya, Vue dirancang agar dapat diadopsi secara bertahap:
- **Sebagai Library**: Anda bisa menggunakan Vue hanya pada bagian kecil dari halaman HTML statis Anda (seperti widget).
- **Sebagai Framework**: Anda bisa menggunakan ekosistem Vue secara penuh (Router, Pinia, CLI) untuk membangun aplikasi skala besar.

## Perbandingan Singkat

Mengapa memilih Vue dibandingkan yang lain?

| Fitur | Vue.js | React | Angular |
| :--- | :--- | :--- | :--- |
| **Kurva Belajar** | Sangat Mudah | Sedang | Sulit |
| **Fleksibilitas** | Tinggi | Sangat Tinggi | Rendah (Opiniated) |
| **Ukuran** | Ringan (~20KB) | Ringan | Berat |

## Kesimpulan

Vue.js adalah pilihan ideal bagi pengembang yang menginginkan kecepatan dalam pengembangan tanpa mengorbankan performa. Dengan dokumentasi yang ramah dan komunitas yang besar, Vue akan sangat membantu Anda membangun aplikasi modern dengan cepat.
