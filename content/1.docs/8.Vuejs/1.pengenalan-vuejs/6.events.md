# Modul 6: Event Handling

Kita dapat menggunakan direktif `v-on` (disingkat menjadi `@`) untuk mendengarkan peristiwa DOM dan menjalankan JavaScript saat peristiwa tersebut dipicu.

## Mendengarkan Event

### 1. Inline Handlers
Cocok untuk logika yang sangat sederhana:
```vue-html
<button @click="count++">Tambah</button>
```

### 2. Method Handlers
Untuk logika yang lebih kompleks, kita memanggil fungsi yang didefinisikan di script:
```vue
<script setup>
function sayHello(event) {
  alert('Halo!')
  // 'event' adalah objek event DOM asli
  if (event) {
    console.log(event.target.tagName)
  }
}
</script>

<template>
  <button @click="sayHello">Sapa</button>
</template>
```

---

## Event Modifiers

Vue menyediakan modifier untuk menangani detail event yang umum (seperti `preventDefault`) sehingga kita bisa fokus pada logika data.

- `.stop`: Berhenti melakukan propagasi (stop propagation).
- `.prevent`: Mencegah aksi default (seperti reload saat submit form).
- `.once`: Event hanya dipicu sekali.
- `.capture`: Menggunakan mode capture.

```vue-html
<!-- Mencegah reload halaman saat submit -->
<form @submit.prevent="onSubmit"></form>

<!-- Klik tidak akan merambat ke elemen induk -->
<a @click.stop="doThis"></a>
```

---

## Key Modifiers (Keyboard Event)

Kita bisa mendengarkan tombol tertentu pada keyboard:

```vue-html
<!-- Hanya jalankan submit jika tombol Enter ditekan -->
<input @keyup.enter="submitForm">

<!-- Mendengarkan kombinasi tombol (Ctrl + Click) -->
<div @click.ctrl="doSomething">Klik dengan Ctrl</div>
```

---

## Praktik Terbaik

::tip{icon="i-lucide-shield-check"}
Selalu gunakan `@click.prevent` pada elemen `<a>` atau `<form>` jika Anda tidak ingin terjadi perpindahan halaman atau reload. Ini sangat penting untuk aplikasi SPA.
::

### Contoh: Menangani Input Form
```vue
<script setup>
import { ref } from 'vue'

const message = ref('')

function handleEnter() {
  if (message.value.trim() === '') return
  alert(`Anda mengetik: ${message.value}`)
  message.value = ''
}
</script>

<template>
  <input 
    v-model="message" 
    @keyup.enter="handleEnter"
    placeholder="Ketik sesuatu lalu tekan Enter"
    class="border p-2 rounded w-full"
  >
</template>
```
