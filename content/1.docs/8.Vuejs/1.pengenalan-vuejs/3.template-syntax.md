# Modul 3: Template Syntax

Vue menggunakan sintaks template berbasis HTML yang memungkinkan Anda untuk secara deklaratif menghubungkan data JavaScript ke tampilan (DOM).

## Interpolasi Teks

Cara paling dasar untuk menampilkan data adalah menggunakan "Mustache" syntax (kurung kurawal ganda).

```vue-html
<span>Pesan: {{ msg }}</span>
```

Nilai di dalam `{{ }}` akan digantikan dengan nilai dari variabel `msg` yang didefinisikan di bagian script.

---

## Direktif (Directives)

Direktif adalah atribut khusus dengan awalan `v-` yang memberikan instruksi khusus kepada Vue.

### 1. v-bind (Pengikatan Atribut)
Digunakan untuk mengikat atribut HTML (seperti `src`, `href`, `class`) ke data JavaScript.

```vue-html
<!-- Sintaks lengkap -->
<img v-bind:src="imageSrc">

<!-- Singkatan (Shorthand) -->
<img :src="imageSrc">
```

### 2. v-on (Event Handling)
Digunakan untuk mendengarkan event dari DOM (seperti klik, input, submit).

```vue-html
<!-- Sintaks lengkap -->
<button v-on:click="doSomething">Klik Saya</button>

<!-- Singkatan (Shorthand) -->
<button @click="doSomething">Klik Saya</button>
```

### 3. v-model (Two-way Binding)
Sangat berguna untuk form input. Data di script dan input di tampilan akan selalu sinkron.

```vue-html
<input v-model="username" placeholder="Masukkan nama">
<p>Halo, {{ username }}!</p>
```

## Rendering Kondisional & List

::card-group
  :::card
  ---
  icon: i-lucide-eye-off
  title: v-if / v-else
  ---
  Menampilkan atau menyembunyikan elemen berdasarkan kondisi boolean.
  ```vue-html
  <p v-if="isLoggedIn">Selamat Datang!</p>
  <p v-else>Silakan Masuk.</p>
  ```
  :::

  :::card
  ---
  icon: i-lucide-list
  title: v-for
  ---
  Melakukan perulangan untuk menampilkan daftar data (array).
  ```vue-html
  <ul>
    <li v-for="item in items" :key="item.id">
      {{ item.text }}
    </li>
  </ul>
  ```
  :::
::

## Menggunakan Ekspresi JavaScript

Di dalam `{{ }}` atau nilai direktif, Anda bisa menggunakan ekspresi JavaScript tunggal:

```vue-html
{{ number + 1 }}
{{ ok ? 'YES' : 'NO' }}
{{ message.split('').reverse().join('') }}
```

::warning{icon="i-lucide-alert-triangle"}
Hanya ekspresi tunggal yang diizinkan. Anda tidak bisa menggunakan pernyataan (seperti `var a = 1`) atau kontrol alur (seperti `if` statement). Gunakan ternary operator sebagai gantinya.
::
