# Modul 5: Class & Style Binding

Kebutuhan umum dalam pengembangan web adalah memanipulasi daftar class elemen dan gaya inline. Karena keduanya adalah atribut, kita dapat menggunakan `v-bind`.

## Binding HTML Class

Kita dapat mengirimkan objek ke `:class` untuk mengaktifkan class secara dinamis.

### 1. Sintaks Objek
```vue-html
<div
  class="static-class"
  :class="{ active: isActive, 'text-danger': hasError }"
></div>
```
- Class `active` akan muncul jika variabel `isActive` bernilai `true`.
- Class `text-danger` akan muncul jika variabel `hasError` bernilai `true`.

### 2. Sintaks Array
Kita juga dapat mengikat `:class` ke sebuah array:
```vue-html
<div :class="[activeClass, errorClass]"></div>
```

---

## Binding Inline Styles

Sama seperti class, kita bisa menggunakan objek atau array untuk `:style`.

### 1. Sintaks Objek
```vue-html
<div :style="{ color: activeColor, fontSize: fontSize + 'px' }"></div>
```
Vue secara otomatis menangani *prefixing* browser untuk properti CSS yang memerlukannya (seperti `transform`).

### 2. Sintaks Array
Sintaks array memungkinkan Anda menerapkan beberapa objek gaya ke elemen yang sama:
```vue-html
<div :style="[baseStyles, overridingStyles]"></div>
```

---

## Contoh Implementasi Nyata

Bayangkan Anda membuat komponen notifikasi:

```vue
<script setup>
import { ref } from 'vue'

const status = ref('success') // Bisa 'success' atau 'error'
</script>

<template>
  <div 
    class="p-4 rounded-lg border"
    :class="{
      'bg-green-100 border-green-500 text-green-700': status === 'success',
      'bg-red-100 border-red-500 text-red-700': status === 'error'
    }"
  >
    Pesan status: {{ status }}
  </div>
  
  <UButton @click="status = 'error'">Ubah ke Error</UButton>
</template>
```

::callout{icon="i-lucide-palette"}
Penggunaan Class Binding sangat disarankan dibandingkan Style Binding jika Anda menggunakan framework CSS seperti Tailwind CSS, karena lebih menjaga konsistensi desain.
::
