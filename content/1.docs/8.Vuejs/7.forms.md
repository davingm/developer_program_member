# Modul 7: Form Input & v-model

Direktif `v-model` memungkinkan kita untuk membuat pengikatan data dua arah (*two-way binding*) pada elemen input formulir, textarea, dan select.

## Penggunaan Dasar

### 1. Teks (Input & Textarea)
```vue-html
<input v-model="message" placeholder="Edit saya">
<p>Pesan adalah: {{ message }}</p>

<textarea v-model="multiline" placeholder="Masukkan banyak teks"></textarea>
```

### 2. Checkbox
```vue-html
<!-- Single Checkbox (Boolean) -->
<input type="checkbox" id="checkbox" v-model="checked">
<label for="checkbox">{{ checked }}</label>

<!-- Multiple Checkboxes (Array) -->
<input type="checkbox" value="Vue" v-model="checkedNames">
<input type="checkbox" value="React" v-model="checkedNames">
```

### 3. Select (Dropdown)
```vue-html
<select v-model="selected">
  <option disabled value="">Silakan pilih satu</option>
  <option>A</option>
  <option>B</option>
  <option>C</option>
</select>
```

---

## Modifiers v-model

Vue menyediakan modifier untuk menyesuaikan perilaku `v-model`:

- **`.lazy`**: Sinkronisasi data setelah event `change` (saat kursor keluar dari input), bukan saat `input` (setiap kali mengetik).
- **`.number`**: Mengonversi input pengguna menjadi angka secara otomatis.
- **`.trim`**: Menghapus spasi di awal dan akhir input secara otomatis.

```vue-html
<!-- Input akan otomatis menjadi tipe Number -->
<input v-model.number="age" type="number">

<!-- Spasi berlebih akan dibersihkan -->
<input v-model.trim="username">
```

## Komponen Form di Nuxt UI

Karena kita menggunakan Nuxt UI, disarankan untuk menggunakan komponen bawaan seperti `UInput`, `UCheckbox`, dan `USelect` yang sudah memiliki gaya premium.

```vue
<script setup>
import { ref } from 'vue'

const formData = ref({
  email: '',
  password: '',
  remember: false
})

function submit() {
  console.log('Mengirim data:', formData.value)
}
</script>

<template>
  <div class="max-w-sm mx-auto space-y-4 p-6 border rounded-xl shadow-sm">
    <UFormGroup label="Email">
      <UInput v-model="formData.email" type="email" icon="i-heroicons-envelope" />
    </UFormGroup>

    <UFormGroup label="Password">
      <UInput v-model="formData.password" type="password" icon="i-heroicons-lock-closed" />
    </UFormGroup>

    <UCheckbox v-model="formData.remember" label="Ingat saya" />

    <UButton block color="primary" @click="submit">Login</UButton>
  </div>
</template>
```

::tip{icon="i-lucide-info"}
`v-model` pada komponen kustom (seperti di Nuxt UI) bekerja dengan cara yang sama persis dengan elemen HTML asli.
::
