# Modul 10: Vue Router

Vue Router adalah pustaka resmi untuk navigasi di Vue.js. Ia memungkinkan kita untuk mengubah tampilan URL tanpa melakukan reload halaman secara penuh.

## Konfigurasi Dasar

Rute biasanya dikonfigurasi dalam folder `src/router/index.js`.

```js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'

const routes = [
  {
    path: '/',
    name: 'home',
    component: Home
  },
  {
    path: '/about',
    name: 'about',
    // Lazy load: Hanya didownload saat dikunjungi
    component: () => import('../views/About.vue')
  }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

---

## Navigasi di Template

Gunakan komponen `<RouterLink>` untuk berpindah halaman dan `<RouterView>` sebagai tempat munculnya halaman tersebut.

```vue [App.vue]
<template>
  <nav class="flex gap-4 p-4 border-b">
    <RouterLink to="/" active-class="text-primary font-bold">Home</RouterLink>
    <RouterLink to="/about" active-class="text-primary font-bold">About</RouterLink>
  </nav>

  <main class="p-4">
    <RouterView />
  </main>
</template>
```

## Rute Dinamis & Parameter

Anda bisa membuat rute yang menerima parameter, misalnya ID pengguna.

```js
// Di router/index.js
{
  path: '/user/:id',
  component: UserProfile
}
```

Cara mengambil parameter tersebut di komponen:

```vue [UserProfile.vue]
<script setup>
import { useRoute } from 'vue-router'

const route = useRoute()
console.log(route.params.id) // Mengambil 'id' dari URL
</script>
```

## Navigasi Programatik

Kadang kita perlu berpindah halaman menggunakan kode (setelah klik tombol atau login berhasil).

```js
import { useRouter } from 'vue-router'

const router = useRouter()

function login() {
  // Lakukan logika login...
  router.push('/dashboard') // Pindah ke dashboard
}
```

::tip{icon="i-lucide-map"}
Gunakan `router.replace('/path')` jika Anda ingin pindah halaman tanpa meninggalkan jejak di riwayat browser (sehingga user tidak bisa klik tombol 'Back').
::
